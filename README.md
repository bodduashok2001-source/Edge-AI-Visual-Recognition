# Real-Time Edge AI Vision

A computer vision project that performs real-time image classification from a live camera feed using TensorFlow and OpenCV. The goal of this project is to explore lightweight deep learning inference and edge AI deployment techniques with optimized models such as TensorFlow Lite and ONNX Runtime.

## Current Features

- Live webcam video capture
- Real-time image classification
- MobileNetV2 pretrained model
- OpenCV integration
- TensorFlow inference

## Planned Features

- TensorFlow Lite inference
- ONNX Runtime support
- FPS benchmarking
- Model quantization
- Object detection
- Edge device optimization

## Tech Stack

- Python
- OpenCV
- TensorFlow
- TensorFlow Lite
- ONNX Runtime
- NumPy
